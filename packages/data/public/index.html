The domain `hv-monsterdb-data.skk.moe` is for hosting daily dump of the HentaiVerse Monster Database, a community-built project for indexing player-created monsters in a pure text online JRPG game, HentaiVerse.

There are two files available under this domain:

- /persistent.json, which contains all known monsters that are in the "persistent" world.
- /isekai.json, which contains all known monsters that are in the "isekai" world.

You can find out more about the project at the GitHub repo: https://hv-monsterdb.skk.moe.

Check out the front-end application at: https://hv-monster.skk.moe/app.
Check out the userscript for in-game integration at: https://github.com/SukkaW/hv-monsterdb-userscript.
